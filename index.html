Donkey Pong
JavaScript/HTML

Donkey Pong is the result of a dumb joke that I made a reality. Originally envisioned as a cross between Pong and Donkey Kong, Donkey Pong is built with HTML Canvas, JavaScript, and a smidgen of jQuery. It also served as a cool intro to game design - I hope you have as much fun playing it as I did building it.

America Online
React/Redux/RubyonRails

America Online is a single page React/Redux application (with a Rails API) built to explore U.S. National Parks, create a trip, and add parks to that trip. Currently in the works are a few features such as maps integration and detailed individual park statistics. For now, feel free to sign up and create the ultimate road trip itinerary.

HuManatee
RubyOnRails/Postgres

Imagine a platform that could connect you with local volunteers that would help each other with tasks - big or small. HuManatee is that very platform, and those volunteers are the "HuManatees." Create a profile to organize and join events in your community. Users have ratings and reviews tied to their profiles to incentivize involvement. This project was great practice on implementing CRUD functionality and dealing with back-end relationships. Check it out!
